# Organic Juicy

Небольшой учебный лендинг про соки.  
Проект сделан на чистом HTML, CSS и JavaScript без сборки.

## Демо

https://sxnizh.github.io/juicy-landing/

## Технологии

- HTML
- CSS
- JavaScript

## Что есть в проекте

- адаптивная сетка на CSS Grid
- бургер-меню для мобильной версии
- слайдер со стрелками
- слайдер с точками
- анимации и hover-эффекты
- форма в футере

## Как запустить

1. Клонировать репозиторий
git clone [https://github.com/sxnizh/juicy-landing.git](https://github.com/sxnizh/juicy-landing.git)

2. Открыть папку проекта
cd juicy-landing

3. Открыть `index.html` в браузере
